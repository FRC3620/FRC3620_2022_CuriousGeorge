# 2022 Controls

## Driver Controls
* A - X Mode
* LT - Intake up/down + roller (clockwise)
* RT - Fire

## Operator Controls

---
* Dpad - Turret left and right
* X - Shooter ("fire up")
* Right Stick - Climber up + down
* A - Climber tilt out
* B - Climber tilt in
