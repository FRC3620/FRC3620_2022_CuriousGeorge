# 2022 Controls

## Driver Controls
* A - X Mode
* X - Reset NavX
* LB - Intake arm up and down plus roller and belt (clockwise)
* RT - Fire

## Operator Controls

* Dpad - Turret left, forward, right, reverse
* X - Shooter ("fire up")
* Right Stick - Climber up + down
* A - Climber tilt out
* B - Climber tilt in
